20190521总结

对象中的this

如果this作为参数传入，那么this就是指向input

此时要注意形参接收时是不允许命名为this的（因为this为关键字）

无参的方法中this

如果直接在search中写this,那么this为将根据其上下文（即他属于谁）来决定，一般指向window

构造函数中的this

this指向当前实例化的具体的对象（谁调用this所在的函数，那么this就指向谁）
比如：你new的wl，那么this指向wl
你new的孙六，那么this指向孙六

> function People(name,age,sex){
    this.name=name;
    this.age=age;
    this.sex=sex;
} 
< undefined 
> var wl=new People("wl",20,"男")
< undefined
> wl
< People{name:"wl",age:20,sex:"男"}
> var sl=new People("孙六",35,"男");
< undefined 
> sl 
< People{name:"孙六",age:35,sex:"男"}

原型方法中的this

谁调用this所在的函数，那么this就指向谁

> Array.prototype.getTot=function(){
    console.log(this);
}
< f(){
    console.log(this);
}
> var r1=[1,2,3,4];
< undefined
> r1.getTot();
  (4)[1,2,3,4]
< undefined
> var r2=["hello",123];
< undefined
> r2.getTot()
  (2)["hello",123]
< undefined

对象方法中的this

对象==普通属性+方法属性

> var obj={
    name:"zs",
    sayName:function(){
        console.log("你好，my name is:"+this.name);
    }
    }
< undefined
> obj.sayName();
  你好，my name is:zs
< undefined

这里的this指向obj(谁调用this所在的函数，那么this就指向谁。)

结论

谁调用方法，this就指向谁
