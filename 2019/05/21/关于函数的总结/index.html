关于函数的总结

什么是函数？作用？函数的声明方式？

使用function关键字定义的一段具有独立作用域，能被反复执行的语句块。
作用：一般用于描述某种功能，实现某种功能
声明方式：（函数重复声明会覆盖）
1:使用function关键字声明
2:利用函数表达式进行赋值声明
注意：函数表达式声明方式，必须先声明后调用否则会报错
3:利用实例化构造函数“Function”声明 var fnName = new Function(“a”,“b”,”return a*b”)
注意：1:这种写法是将参数列表和函数体放在了一起同样作为了参数。
2:如果只有一个参数，那么这个参数就是函数体（就是花括号里面的代码）。
3:构造函数内的参数无论有多少个，始终会将最后一个参数作为函数体去执行。
4:参数和函数体的语句要作为字符串去呈现。

函数的返回值

1.函数返回值没有设置return 返回值默认是undefined
2.return之后的返回值只能有一个，如果尝试返回多个值，得到的结果始终是最后一个值。
3.如果真的需要函数返回多个值，可以将值组合成一个对象或者数组返回。

参数

由形参和实参构成

对位传参

1.形参和实参是通过位置而不是参数名来匹配对应的，所以传递时位置不能更换必须一一对应
2.形参和实参的个数可以不一致
3.如果形参没有接受到实参传递来的值，那么他为undefined

对象传参

传递参数时，不用考虑参数位置必须是一一对应的方式传递，
通过对象穿参，通过属性名key来匹配，此时就不用考虑位置问题

参数的默认值

Es6写法 （在参数列表里赋值）

1.function fn(name=”章三”，sex=”男”){}
2.function fn(obj={name:”章三”,sex:”男”})

Es5写法 （三元运算符/逻辑”||”）

注意：对于0，-0，null，“”，undefined,NaN逻辑运算结果全为flase.*
1.三元运算符
function fn(a,b){
a=(a==undefined?”章三”:a);
b=(b==undefined?”男”:b);
console.log(a,b)
}
2.逻辑”||”
function fn(a,b){
a=a||”aaa”;
b=b||”bbb”;
console.log(a,b);
}

Arguments

1.代表一个函数所有的参数，是一个类似数组的东西。
2.可以通过下标去值。（在不知道具体的参数个数时，他尤为重要)

将arguments转换成一个数组

1.修改this指向 arguments = Array.prototype.slice.call(arguments);
2.Array.from(arguments)

函数的作用域

Es5中只有 全局作用域 函数作用域
Es6中只有 全局作用域 函数作用域 块级作用域（if判断、for循环等）

关于let的一些说明

1.不允许重复声明
2.不会进行变量提升
3.具有块级作用域

不同作用域变量的访问问题

1.访问变量时，先在当前作用域查找，然后再去上层作用域查找（就近原则）***
2.访问优先级：里层能访问外层，外层不能访问里层

递归（函数自己调用自己）*

1.可以用更少的代码完成很多 需要用大量代码来实现的红能。
2.递归和循环很相似，使用时注意要有结束条件避免死循环。
3.建议优先使用循环代替递归。防止有些浏览器对递归迭代周期过长而产生的报错。

自执行匿名函数 （立即调用的函数表达式）

不需要调用 ，立即执行
优点：防止变量名冲突.***
1.（function(){})()
2.(function(){}())
注意：它也可以有函数名，只能在函数内部调用
例如:(function fn(){return “我是一个即时执行的”+ typeof fn})();

回调函数 （将函数作为参数，传入另一个函数就是回调函数）*

理解：例如有两个函数A和B，如果函数A作为B的一个参数传入B的话，那么A就是回调函数。
例题：
第一步：function fn(a,b){
if(a==”f”) b();
else console.log(b);
};
第二步：function ff(){
alert(“我是回调函数，我会作为一个参数传入到其他函数里面”)
}；
第三步：fn(“f”,ff)

闭包 （返回一个函数的函数）*

当我们想要从外部读取函数里面定义的局部变量时，可以定义一个闭包来实现。
例题：
function fn(){
var n=10;
return function (){
return n;
}
}

闭包的作用

1.可以读取整个父级作用域函数内部的变量
2.让这些变量始终保存在内存中。

闭包的注意事项

1.闭包会使得函数中的变量都被保存在内存中，内存消耗很大，不能滥用。
2.解决方法：在退出函数之前，将不使用的局部变量全部删除。

闭包与let之块级作用域

例题：
var array=[];
for(var i=0;i<10;i++){
array[i]=(function(j){
return function(){
return j;
}
})(j)
}
